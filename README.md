## About

This repository contains DFIR (Digital Forensics and Incident Response) writeups for the challenges I created for the **GCDxN7 CTF**.

## Challenges

The following challenges are documented in this repository:

- **Ping Pong** - DFIR Challenge
- **White Whale** - DFIR Challenge  
- **Goyslop** - DFIR Challenge

## Challenge Writeups

Each challenge writeup includes:
- Challenge description and objectives
- Tools and techniques used
- Step-by-step solution methodology
- Flags and key findings
